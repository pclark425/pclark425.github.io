Theory (General Information)

ID: theory-904

Name: Hierarchical Episodic-Semantic Memory Integration for LLM Agents in Text Games

Type: general

Theory Query: Build a theory of how LLM agents for text games can best use memory to solve text game tasks.

Description: This theory posits that LLM agents achieve optimal performance in text games by dynamically integrating hierarchical episodic (event-based) and semantic (fact-based) memory representations. The agent constructs and maintains a multi-level memory structure, where low-level episodic traces (specific actions and observations) are abstracted into higher-level semantic knowledge (rules, object properties, world models), and retrieval is contextually modulated to balance specificity and generalization. This enables efficient reasoning, planning, and adaptation to novel game scenarios.

Theory (Details)

Theory Statements

Statement 0: Hierarchical Memory Abstraction (qualitative)

ConditionDetails
IF
• LLM agent → encounters → sequence of events in text game
THEN
• LLM agent → abstracts → episodic traces into semantic knowledge at multiple levels, and
• LLM agent → stores → both episodic and semantic representations in memory

Supporting Evidence for this Law

  1. Hierarchical memory structures are effective in human cognition and AI for balancing detail and generalization.
  2. Semantic memory enables transfer and abstraction, while episodic memory supports detailed recall in complex tasks.
  3. Hierarchical RL and memory-augmented neural networks show improved performance in long-horizon, partially observable tasks.

Existing Law Comparison

Likely Classification: somewhat-related-to-existing

Explanation: While hierarchical memory is known, its explicit, adaptive integration in LLM agents for text games is novel.

What Already Exists: Hierarchical memory and abstraction are established in cognitive science and some AI models.

What is Novel: Dynamic, context-sensitive integration of episodic and semantic memory in LLM agents for text games.

References:

  • Tulving (1972) Episodic and semantic memory [distinction in human memory]
  • Botvinick et al. (2019) Reinforcement learning, fast and slow [hierarchical RL]
  • Graves et al. (2016) Hybrid computing using a neural network with dynamic external memory [memory-augmented networks]

Statement 1: Contextual Memory Retrieval Modulation (qualitative)

ConditionDetails
IF
• LLM agent → faces → decision point in text game, and
• current context → matches → semantic or episodic memory cues
THEN
• LLM agent → retrieves → most relevant memory representation(s) for reasoning

Supporting Evidence for this Law

  1. Contextual retrieval is critical for flexible reasoning in both humans and AI agents.
  2. Memory-augmented transformers and retrieval-augmented LLMs show improved performance when retrieval is context-sensitive.

Existing Law Comparison

Likely Classification: closely-related-to-existing

Explanation: The general principle is known, but its hierarchical, adaptive application in LLM text game agents is novel.

What Already Exists: Contextual retrieval is established in cognitive science and retrieval-augmented models.

What is Novel: Hierarchical, context-sensitive retrieval modulation in LLM agents for text games.

References:

  • Kumar et al. (2016) Ask Me Anything: Dynamic Memory Networks for NLP [contextual memory retrieval]
  • Lewis et al. (2020) Retrieval-augmented generation for knowledge-intensive NLP tasks [retrieval-augmented LLMs]

New Predictions (Likely outcome)

  1. LLM agents with hierarchical episodic-semantic memory will outperform agents with flat or single-level memory on long-horizon, complex text games.
  2. Context-sensitive retrieval will enable more efficient planning and fewer irrelevant memory accesses.

New Predictions (Unknown outcome/high-entropy)

  1. Emergent semantic abstractions may enable zero-shot transfer to novel games with similar structure.
  2. Over-abstraction may cause loss of critical episodic details, leading to unexpected failures in games with subtle dependencies.

Negative Experiments

  1. If flat memory architectures outperform hierarchical ones in complex text games, the theory would be challenged.
  2. If context-insensitive retrieval yields better results than contextual retrieval, the theory's assumptions may be flawed.

Unaccounted for Evidence

  1. Some text games may require purely episodic recall due to highly unique, non-generalizable events.
