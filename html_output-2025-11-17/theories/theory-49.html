Theory (General Information)

ID: theory-49

Name: Memory Modality-Task Alignment Theory

Type: general

Theory Query: Build a theory of how agents can use memory to help solve text games, based on the following results.

Description: The effectiveness of different memory types in text-based games is determined by alignment between the memory's structural properties and the task's cognitive demands. Specifically: (1) graph-based memory excels at spatial reasoning and entity tracking, (2) episodic memory excels at procedural recall and instruction following, (3) recurrent memory excels at short-term context integration, and (4) reflection-based memory excels at strategy refinement across attempts. Misalignment between memory type and task demands results in sub-optimal performance regardless of memory capacity.

Theory (Details)

Theory Statements

  1. Memory effectiveness follows a task-alignment hierarchy: perfectly aligned memory provides 40-60% improvement, partially aligned memory provides 10-30% improvement, misaligned memory provides <5% improvement or negative impact.
  2. Spatial/navigational tasks (e.g., Cleaning, Treasure Hunt) require graph-based memory for >50% success rate; episodic-only memory achieves <30% success rate on these tasks.
  3. Procedural/sequential tasks (e.g., Cooking, recipe following) require episodic memory for >70% success rate; semantic-only memory achieves <50% success rate on complex procedural tasks.
  4. Multi-attempt learning tasks benefit from reflection memory, with optimal performance at 2-4 attempts; beyond 4 attempts, reflection memory shows diminishing or negative returns.
  5. Short-term context integration tasks (within 5-10 steps) benefit from recurrent memory; long-term dependencies (>20 steps) require explicit external memory (graph or episodic) for >60% success rate.
  6. The performance gap between aligned and misaligned memory grows with task complexity: for simple tasks (<20 steps), gap is <20%; for complex tasks (>50 steps), gap exceeds 50%.

Supporting Evidence

  1. AriGraph without episodic memory maintains high performance on navigation-heavy Cleaning (0.92) but drops substantially on procedural Cooking (0.64 Medium, 0.45 Hard) [e238.1]
  2. Reflexion shows marked performance growth at second try but degrades with subsequent tries, suggesting reflection memory is optimized for few-shot adaptation [e238.6]
  3. Full history baseline performs very poorly on Cleaning (0.05) but better on Cooking (0.18), suggesting unstructured memory is particularly bad for spatial tasks [e238.3]
  4. RAG baseline shows relatively balanced but mediocre performance across tasks (0.33-0.39), suggesting vector retrieval doesn't align well with any specific task type [e238.5]
  5. Simulacra memory (scored episodic) performs well on Cleaning (0.70) but poorly on Treasure Hunt (0.40) and Cooking (0.30) [e238.7]
  6. LSTM-DRQN with recurrent memory shows modest improvements (3.9%) over non-recurrent baseline, suggesting recurrent memory provides limited benefits for long-horizon tasks [e236.3]
  7. Sweet&Sour with positive+negative reflection memory substantially outperforms failure-only Reflexion (54.6 vs 45.3 for GPT-4o) [e232.0] [e232.1]
  8. Commonsense-enhanced agents with external knowledge bases help with thematic reasoning and affordance understanding [e229.2]

New Predictions (Likely outcome)

  1. In a new text game requiring pure spatial reasoning (navigating a maze with no procedural elements), graph-based memory will achieve >80% success rate while episodic-only memory will achieve <40%.
  2. For a task requiring memorization and recall of a 15-step procedure with no navigation, episodic memory will achieve >75% success rate while graph-only memory will achieve <45%.
  3. In a task requiring both navigation and procedures but with clear separation (navigate first, then execute procedure), hybrid memory will show exactly additive benefits of each component with <5% synergy.

New Predictions (Unknown outcome/high-entropy)

  1. For tasks requiring temporal reasoning about event sequences and causality (e.g., 'what caused the door to open?'), it's unclear whether episodic memory (preserving temporal order) or graph memory (capturing causal relations) would be more effective - difference could range from -30% to +40%.
  2. In tasks requiring analogical reasoning across different game scenarios, it's unknown whether abstract semantic memory or concrete episodic examples would be more beneficial - performance difference unclear.
  3. For tasks with nested sub-goals (goals within goals), the optimal memory structure (hierarchical graph vs. flat episodic vs. hybrid) is unknown and could show 20-50% performance variation.

Negative Experiments

  1. Finding tasks where misaligned memory consistently outperforms aligned memory would challenge the alignment principle.
  2. Demonstrating that a single memory type (e.g., graph-based) achieves optimal performance across all task types would challenge the task-specificity claim.
  3. Showing that memory alignment provides no performance benefit in complex tasks (>50 steps) would challenge the complexity-scaling claim.
  4. Finding that reflection memory continues to improve performance beyond 10 attempts would challenge the diminishing returns claim.

Unaccounted for Evidence

  1. How to automatically determine the optimal memory type for a new task without manual analysis is not addressed [e238.0] [e238.1] [e238.2]
  2. The mechanisms for dynamically switching between memory types during task execution are not explored [e238.0]
  3. How memory alignment interacts with different LLM architectures and sizes is not systematically studied [e232.0] [e238.0]
