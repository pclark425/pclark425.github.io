Theory (General Information)

ID: theory-1661

Name: Theory of Representational Fidelity in Tool-Augmented LLM Simulation

Type: general

Theory Query: Build a theory of factors affecting the accuracy of using large language models (LLMs) as text-based simulators for specific scientific subdomains.

Description: This theory asserts that the fidelity with which external tools represent the formal structures, entities, and processes of a scientific subdomain is the primary determinant of LLM-based simulation accuracy. High-fidelity tools that closely mirror the subdomain's ontologies and computational rules enable LLMs to simulate with high accuracy, while low-fidelity or approximate tools introduce systematic errors and limit performance.

Statement 0: Representational Fidelity Law (qualitative)

ConditionDetails
IF
• external tool → has_high_fidelity_representation_of → scientific subdomain's formal structures, and
• LLM → is_augmented_with → external tool
THEN
• LLM+tool system → achieves_high_accuracy_on → simulation tasks in that subdomain

Supporting Evidence for this Law

  1. LLMs with access to quantum chemistry engines or symbolic math solvers achieve expert-level accuracy on corresponding tasks.
  2. Physics engines that accurately encode Newtonian or quantum rules enable high-fidelity simulation.

Statement 1: Low-Fidelity Tool Limitation Law (qualitative)

ConditionDetails
IF
• external tool → has_low_fidelity_representation_of → scientific subdomain's formal structures, and
• LLM → is_augmented_with → external tool
THEN
• LLM+tool system → is_limited_by → systematic errors from tool's representation, and
• LLM+tool system → achieves_lower_accuracy_on → simulation tasks in that subdomain

Supporting Evidence for this Law

  1. LLMs using approximate calculators or heuristic-based tools for complex scientific tasks show systematic errors and lower accuracy.
  2. Empirical studies show that low-fidelity tools introduce domain-specific biases and errors.

Likely Classification: new

New Predictions (Likely outcome)

  1. Upgrading a tool to a higher-fidelity representation (e.g., from heuristic to formal symbolic engine) will increase LLM simulation accuracy.
  2. Introducing systematic errors into a tool's representation will cause corresponding errors in LLM simulation outputs.

New Predictions (Unknown outcome/high-entropy)

  1. If an LLM can learn to compensate for low-fidelity tool errors via meta-reasoning, the impact of tool fidelity may be reduced.
  2. Hybrid tools that combine high- and low-fidelity representations may yield unpredictable accuracy outcomes.

Negative Experiments

  1. If LLMs achieve high simulation accuracy with low-fidelity tools, the theory is challenged.
  2. If increasing tool fidelity does not improve simulation accuracy, the theory is challenged.

Unaccounted for Evidence

  1. Cases where LLMs internally correct for systematic tool errors.
