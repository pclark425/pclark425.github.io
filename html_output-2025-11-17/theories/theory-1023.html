Theory (General Information)

ID: theory-1023

Name: Entity-Centric Memory Graphs Enable Robust Object Tracking and Manipulation

Type: specific

Theory Query: Build a theory of how LLM agents for text games can best use memory to solve text game tasks.

Description: This theory proposes that memory modules structured as entity-centric graphs—where nodes represent objects, locations, and agents, and edges represent relations and actions—are particularly effective for tracking and manipulating objects in text games. Such memory graphs allow LLM agents to maintain up-to-date world models, resolve coreference, and plan object interactions, even under partial observability.

Theory (Details)

Theory Statements

Statement 0: Entity Graph Memory Supports Coreference and State Tracking (qualitative)

ConditionDetails
IF
• agent → has_memory_module → entity-centric graph, and
• text game → contains → multiple objects and locations
THEN
• agent → can_resolve → coreference and object state changes, and
• agent → can_track → object locations and properties over time

Supporting Evidence for this Law

  1. Graph-based memory modules have been shown to improve object tracking and manipulation in text games with many entities.
  2. Entity-centric graphs allow agents to update and query the state of objects even when not directly observed.
  3. Coreference resolution in text games is improved when agents maintain explicit representations of entities and their relations.
  4. Partial observability in text games often leads to ambiguous references, which can be disambiguated by maintaining a structured memory of entities and their states.

Existing Law Comparison

Likely Classification: closely-related-to-existing

Explanation: While entity graphs are used, their necessity and sufficiency for robust tracking is a novel, formalized claim.

What Already Exists: Graph-based memory has been used for object tracking in text games.

What is Novel: The law formalizes the sufficiency and necessity of entity-centric graph memory for robust coreference and state tracking in LLM agents.

References:

  • Ammanabrolu & Hausknecht (2020) Graph-based Memory for Text-based Games [Uses entity graphs for memory, but not formalized as necessary/sufficient]
  • Ammanabrolu et al. (2020) Graph Constrained Reinforcement Learning for Text Games [Graph memory for planning, but not formalized as essential]

Statement 1: Entity Graph Memory Enables Efficient Object Manipulation (qualitative)

ConditionDetails
IF
• agent → has_memory_module → entity-centric graph, and
• agent → uses_memory → to inform action selection
THEN
• agent → achieves → higher success rates in object manipulation tasks

Supporting Evidence for this Law

  1. Agents with entity graph memory solve object manipulation puzzles more efficiently than those with flat or unstructured memory.
  2. Structured memory allows agents to plan multi-step object interactions by maintaining up-to-date knowledge of object states and locations.
  3. Empirical results show that agents with explicit entity tracking outperform those relying solely on LLM context windows in complex manipulation tasks.

Existing Law Comparison

Likely Classification: closely-related-to-existing

Explanation: The necessity claim is a novel, formalized extension of prior empirical findings.

What Already Exists: Entity graph memory has been shown to help with object manipulation.

What is Novel: The law asserts that entity-centric graph memory is necessary for efficient object manipulation in complex text games.

References:

  • Ammanabrolu & Hausknecht (2020) Graph-based Memory for Text-based Games [Entity graph memory for object manipulation, but not formalized as necessary]

New Predictions (Likely outcome)

  1. LLM agents with entity-centric graph memory will outperform those with flat or list-based memory on tasks requiring object tracking and manipulation.
  2. Entity graph memory will enable agents to resolve ambiguous references (e.g., 'it', 'the key') more accurately.
  3. Agents with entity-centric graph memory will require fewer steps to solve multi-object puzzles than those without such memory.

New Predictions (Unknown outcome/high-entropy)

  1. Entity-centric graph memory may enable agents to generalize to new object types or relations not seen during training.
  2. Such memory structures may facilitate transfer to non-textual environments with similar relational structure.
  3. Entity-centric graph memory may allow for zero-shot adaptation to novel game mechanics involving new types of relations.

Negative Experiments

  1. If agents with entity-centric graph memory do not outperform those with unstructured memory on object tracking tasks, the theory is called into question.
  2. If coreference resolution does not improve with entity graph memory, the sufficiency claim is weakened.
  3. If agents with entity-centric graph memory fail to plan multi-step object manipulations more efficiently than baselines, the necessity claim is challenged.

Unaccounted for Evidence

  1. Some text games with very few objects or simple state transitions may not benefit from entity-centric graph memory.
