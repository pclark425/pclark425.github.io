Theory (General Information)

ID: theory-891

Name: Hierarchical Memory Organization for Compositional Task Solving

Type: general

Theory Query: Build a theory of how language model agents can best use memory to solve tasks.

Description: This theory asserts that language model agents can best solve complex, compositional tasks by organizing memory into hierarchical structures, where higher-level abstractions guide retrieval and recombination of lower-level episodic or semantic memories. This enables efficient generalization, transfer, and flexible adaptation to novel task configurations.

Supporting Evidence for this Law

  2. Hierarchical reinforcement learning agents generalize better to novel task compositions.
  3. Language models with hierarchical memory modules show improved transfer and recombination abilities.

Statement 1: Schema-Guided Memory Recombination Law (qualitative)

ConditionDetails

• agent → retrieves → high-level_schema, and

  1. Humans use schemas to recombine memory fragments for novel problem solving.

  3. Language models with compositional memory modules show improved zero-shot generalization.
