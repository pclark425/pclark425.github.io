Theory (General Information)

ID: theory-502

Name: Heterogeneous Multi-Format Proxy Network Theory

Type: general

Theory Query: Build a theory of the representational format of conceptual knowledge in brains at a functional level, based on the following results.

Description: Conceptual knowledge in brains is functionally represented as a dynamic, context-sensitive network of co-referring representational formats (prototypes, exemplars, theory-like/causal structures, and symbolic/classical definitions), each stored in long-term memory and selectively activated ('proxyfied') in working memory according to task demands, context, and similarity thresholds. This network is supported by distributed, modality-specific neural ensembles, amodal/multimodal hubs, and symbolic/graphical structures, with arbitration and integration governed by dual-process (Type 1/Type 2) mechanisms. The representational format is thus inherently heterogeneous, with flexible selection and integration of multiple representational bodies for each concept.

Statement 0: Heterogeneous Proxy Activation Law (qualitative)

ConditionDetails
IF
• concept → is_stored_as → multiple_representational_bodies (prototype, exemplar, theory-like, classical, perceptual-symbol, distributional, symbolic, neural, etc.), and
• task → has_contextual_requirements → context
THEN
• concept → activates_in_working_memory → the_representational_body_most_relevant_to_context

Supporting Evidence for this Law

  1. Heterogeneous proxytypes theory and Dual-PECCS implementation show that concepts are stored as multiple co-referring bodies and context-sensitive tokenization selects the representation(s) that 'proxy' for the concept in working memory. [e3472.5] [e3623.5] [e3459.8] [e3472.4] [e3623.4]
  2. Empirical findings: different tasks elicit prototype, exemplar, or theory-based reasoning; implemented in systems (DUAL-PECCS) where prototypes and exemplars coexist and are accessed according to similarity and contextual relevance. [e3472.5] [e3623.5] [e3472.2] [e3472.1] [e3623.1] [e3623.0]
  3. Machery's heterogeneity argument: concepts are not a natural kind, and multiple representational kinds (prototype, exemplar, theory, perceptual-symbol, etc.) coexist and are used depending on context. [e3614.2] [e3614.3] [e3614.4] [e3614.5]

New Predictions (Likely outcome)

  1. In tasks requiring rapid, typicality-based categorization, neural activity will show greater engagement of distributed modality-specific ensembles and prototype/exemplar retrieval circuits.
  2. When a categorization task involves exceptions or atypical items, there will be increased activation in symbolic/Type 2 regions (e.g., prefrontal cortex) and longer reaction times.
  3. Lesions to modality-specific areas will selectively impair concrete concept processing, while lesions to hub regions will produce broader, multimodal conceptual deficits.
  4. Tasks that require theory-like or causal reasoning (e.g., property induction based on hidden features) will show increased reliance on theory-like representational bodies and associated neural substrates.
  5. When a stimulus is highly similar to a stored exemplar, categorization will be faster and more accurate, and will rely less on prototype or rule-based processes.

New Predictions (Unknown outcome/high-entropy)

  1. In novel, ambiguous categorization tasks, the brain will dynamically arbitrate between multiple representational formats, and the pattern of activation will predict individual differences in strategy selection.
  2. If a new representational format (e.g., a hybrid neuro-symbolic code) is introduced via training, the system will integrate it as an additional proxy, and selection will depend on its contextual utility.
  3. In patients with selective working memory impairment, the process of proxyfication (activation of the relevant representational body) will be disrupted, leading to context-inappropriate categorization errors.
  4. If a concept is learned exclusively through language (e.g., in the congenitally blind), the system will rely more on symbolic or distributional proxies, and less on modality-specific proxies, but will still integrate with the proxy network.
  5. In tasks requiring rapid switching between contexts (e.g., task switching with changing category boundaries), the system will show flexible re-proxyfication, with measurable neural and behavioral signatures of representational switching.

Negative Experiments

  1. If it is found that only a single representational format (e.g., prototype or symbolic) is ever activated for any concept across all tasks and contexts, this would challenge the theory.
  2. If dual-process arbitration (Type 1/Type 2) is not observed in neural or behavioral data (e.g., no dissociation in reaction times or error patterns), the theory would be called into question.
  3. If context does not modulate which representational body is activated (i.e., proxyfication is invariant to task demands), the theory would be undermined.
  4. If patients with selective damage to one representational format (e.g., modality-specific cortex) do not show corresponding deficits in tasks that should rely on that format, the theory would be challenged.
  5. If introducing a new representational format (e.g., a new hybrid code) does not result in its integration into the proxy network or does not affect categorization behavior, the theory would be called into question.

Unaccounted for Evidence

  1. The precise neural mechanisms for how proxyfication (selection and activation of the relevant representational body) is implemented and coordinated across distributed and hub regions remain unspecified. [e3472.5] [e3623.5] [e3472.9] [e3472.4] [e3472.8] [e3472.10] [e3623.8]
  2. The theory does not specify how new representational formats (e.g., program-induction, neuro-symbolic hybrids, conceptual spaces, DnS-based vector spaces) are learned and integrated into the proxy network. [e3450.3] [e3450.0] [e3476.0] [e3463.1] [e3463.0]
  3. The theory does not fully account for the mechanisms by which arbitration between conflicting proxies is resolved, especially in cases of ambiguous or conflicting evidence. [e3472.9] [e3472.10] [e3623.8]
