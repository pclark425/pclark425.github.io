Theory (General Information)

ID: theory-47

Name: Hybrid Memory Complementarity Theory

Type: general

Theory Query: Build a theory of how agents can use memory to help solve text games, based on the following results.

Description: Optimal performance in text-based games requires hybrid memory systems that combine semantic memory (structured knowledge graphs capturing entity-relation facts) with episodic memory (raw observational records) because different task components require different memory modalities: navigation and spatial reasoning benefit from semantic graphs, while procedural recall and instruction-following require episodic detail. The synergy between these memory types exceeds the sum of their individual contributions.

Theory (Details)

Theory Statements

  1. Task performance is maximized when memory modality matches task requirements: semantic graphs for spatial/relational reasoning, episodic records for procedural/sequential reasoning.
  2. The performance gain from hybrid memory is super-additive: hybrid memory performance > semantic-only performance + episodic-only performance - baseline, due to synergistic retrieval where episodic edges link to relevant semantic facts.
  3. For navigation-dominant tasks, semantic memory contributes 80-90% of performance; for procedural-dominant tasks, episodic memory contributes 50-70% of performance beyond semantic baseline.
  4. Episodic memory becomes increasingly critical as task complexity increases: the performance gap between hybrid and semantic-only memory grows with the number of procedural steps required.
  5. Hybrid memory systems require explicit linking mechanisms (e.g., hyperedges connecting co-occurring facts) to achieve synergy; independent semantic and episodic stores without links provide minimal advantage over single-modality memory.

Supporting Evidence

  1. AriGraph with full semantic+episodic memory achieves perfect scores (1.0) on Cooking tasks, while semantic-only variant drops to 0.64 (Medium) and 0.45 (Hard) [e238.0] [e238.1]
  2. Episodic memory component substantially improves performance on tasks requiring detailed observation recall (Cooking), while semantic graph alone supports navigation [e238.1]
  3. AriGraph without episodic memory shows high performance on navigation-heavy Cleaning (0.92) but degraded performance on procedural Cooking tasks [e238.1]
  4. AriGraph's hybrid memory enables near-oracle NetHack performance (593 score) with restricted observations, approaching full-observation baseline (675) [e238.0] [e238.8]
  5. Semantic memory alone is sufficient for navigation-heavy tasks but lacks detailed step recall needed for procedural tasks [e238.1]
  6. RAG baseline (vector retrieval only) underperforms AriGraph across all tasks, showing unstructured retrieval is insufficient [e238.5]
  7. Summarization baseline loses fine-grained episodic detail, performing worse than hybrid memory on procedural tasks [e238.4]

New Predictions (Likely outcome)

  1. In a new text game requiring both navigation through a 20+ room environment AND following a 10-step recipe, hybrid memory will show >40% improvement over semantic-only and >30% improvement over episodic-only approaches.
  2. For tasks with interleaved navigation and procedural steps (e.g., 'go to kitchen, chop vegetables, go to pantry, get spices, return to kitchen, cook'), hybrid memory will show >50% improvement over either single-modality approach.
  3. In games where procedural instructions are given once at the beginning and must be recalled later, episodic memory will be essential: semantic-only approaches will achieve <30% success rate while hybrid approaches will achieve >80%.

New Predictions (Unknown outcome/high-entropy)

  1. In text games requiring counterfactual reasoning ('what would have happened if I had taken the other path?'), it's unclear whether episodic memory of alternative branches or semantic abstraction of causal rules would be more beneficial - performance difference could range from -20% to +40%.
  2. For games with unreliable narrators or contradictory information, maintaining both semantic (integrated beliefs) and episodic (raw contradictory observations) memory might enable better reasoning, but could also cause confusion - net effect unknown.
  3. In highly procedural tasks where the same procedure must be executed many times with slight variations, it's unclear whether episodic memory of each execution helps (providing examples) or hurts (causing interference) - could show anywhere from -15% to +25% impact.

Negative Experiments

  1. Finding tasks where semantic-only memory consistently outperforms hybrid memory would challenge the complementarity claim.
  2. Demonstrating that episodic and semantic memory provide exactly additive benefits (no synergy) would challenge the super-additive interaction claim.
  3. Showing that in purely procedural tasks with no navigation, semantic memory provides no benefit over episodic-only would challenge the universal value of semantic structure.
  4. Finding that unlinking episodic and semantic memory (storing them independently) provides the same performance as linked hybrid memory would challenge the importance of explicit connection mechanisms.

Unaccounted for Evidence

  1. The optimal ratio of semantic to episodic memory capacity for different task types is not characterized [e238.0] [e238.1]
  2. How to handle conflicts between semantic beliefs and episodic observations is not fully addressed [e238.0]
  3. The computational cost tradeoffs between maintaining both memory types versus single-modality memory are not quantified [e238.0]
