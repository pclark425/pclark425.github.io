Theory (General Information)

ID: theory-771

Name: Distributed Symbolic-Subsymbolic Hybrid Representation for Arithmetic in LLMs

Type: general

Theory Query: Build a theory of how language models perform arithmetic.

Description: This theory proposes that LLMs encode arithmetic knowledge through a hybrid of distributed (subsymbolic) and symbolic representations. Arithmetic operations are not stored as explicit rules, but as patterns in high-dimensional vector spaces, which can be dynamically composed and manipulated via attention and contextual cues. Chain-of-thought and program synthesis act as mechanisms for activating and sequencing these latent representations, enabling flexible and generalizable arithmetic reasoning.

Theory Statements

Statement 0: Latent Vector Encoding of Arithmetic Operations (qualitative)

ConditionDetails
IF
• LLM → is_trained_on → text_with_arithmetic_examples
THEN
• LLM → encodes → arithmetic_operations_as_high_dimensional_vectors

Supporting Evidence for this Law

  1. Probing studies show arithmetic facts and procedures are recoverable from LLM activations.
  2. LLMs can perform arithmetic even when explicit symbolic rules are not present in the prompt.

Existing Law Comparison

What Already Exists: Distributed representations in neural networks are well-established.

What is Novel: This law applies distributed encoding specifically to arithmetic operations in LLMs.

Statement 1: Dynamic Composition via Attention and Context (qualitative)

ConditionDetails
IF
• LLM → receives → arithmetic_problem_with_contextual_cues
THEN
• LLM → dynamically_composes → latent_arithmetic_representations, and
• LLM → generates → stepwise_or_direct_solution

Supporting Evidence for this Law

  1. Attention patterns in LLMs shift to relevant tokens and intermediate results during arithmetic reasoning.
  2. LLMs can flexibly switch between direct recall and stepwise computation depending on context.

Existing Law Comparison

Likely Classification: closely-related-to-existing

Explanation: The law extends known mechanisms to the specific case of arithmetic reasoning.

What Already Exists: Attention-based composition is a core mechanism in transformers.

What is Novel: This law frames attention as a means for dynamic symbolic-subsymbolic composition in arithmetic tasks.

References:

  • Vaswani et al. (2017) Attention is All You Need [Attention mechanism]
  • Geva et al. (2021) Transformer Feed-Forward Layers Are Key-Value Memories [Dynamic memory in transformers]

New Predictions (Likely outcome)

  1. Probing LLM activations during arithmetic tasks will reveal compositional patterns corresponding to arithmetic operations.
  2. Altering attention patterns (e.g., via prompt engineering) will change the model's arithmetic reasoning strategy.

New Predictions (Unknown outcome/high-entropy)

  1. It may be possible to directly manipulate latent vectors to induce novel arithmetic operations.
  2. Hybrid symbolic-subsymbolic representations may enable LLMs to invent new arithmetic algorithms not present in training data.

Negative Experiments

  1. If LLMs cannot be shown to encode arithmetic knowledge in distributed activations, the theory is weakened.
  2. If attention does not mediate dynamic composition during arithmetic, the theory's mechanism is challenged.

Unaccounted for Evidence

  1. The theory does not explain catastrophic failures on out-of-distribution arithmetic tasks.
