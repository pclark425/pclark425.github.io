Theory (General Information)

ID: theory-936

Name: Hierarchical Episodic-Semantic Memory Integration for LLM Text Game Agents

Type: general

Theory Query: Build a theory of how LLM agents for text games can best use memory to solve text game tasks.

Description: This theory posits that LLM agents achieve optimal performance in text games by dynamically integrating two distinct but interacting memory systems: episodic memory (detailed, temporally ordered records of past actions and observations) and semantic memory (abstracted, generalized knowledge about the game world, rules, and object properties). The agent must learn when to retrieve, update, and synthesize information from both systems to support planning, inference, and adaptation to novel situations.

Theory (Details)

Theory Statements

Statement 0: Episodic-Semantic Memory Synergy Law (qualitative)

ConditionDetails
IF
• LLM agent → engages_in → text game task, and
• task → requires → multi-step reasoning or adaptation
THEN
• LLM agent → retrieves_and_integrates → episodic and semantic memory traces

Supporting Evidence for this Law

  1. Human cognition relies on both episodic and semantic memory for complex problem solving.
  2. LLM agents with both memory types outperform those with only one in text-based tasks.
  3. Semantic memory enables generalization, while episodic memory supports context-sensitive recall.

Existing Law Comparison

Likely Classification: somewhat-related-to-existing

Explanation: While dual-memory systems exist in theory and some AI, their dynamic, context-driven integration in LLM text game agents is new.

What Already Exists: Dual-memory systems are well-established in cognitive science and some AI architectures.

What is Novel: The explicit, dynamic integration of these systems in LLM agents for text games is novel.

References:

  • Tulving (1972) Episodic and semantic memory [dual-memory theory in humans]
  • Weston et al. (2015) Memory Networks [memory-augmented neural networks]
  • Ammanabrolu et al. (2020) Graph-based Memory for Interactive Fiction Games [episodic/semantic memory in text game agents]

Statement 1: Adaptive Memory Update Law (qualitative)

ConditionDetails
IF
• LLM agent → detects → novel or unexpected game state
THEN
• LLM agent → updates → episodic and/or semantic memory representations

Supporting Evidence for this Law

  1. Humans update both episodic and semantic memory in response to novel events.
  2. Agents that adapt memory representations to new information perform better in dynamic environments.

Existing Law Comparison

Likely Classification: somewhat-related-to-existing

Explanation: The principle is known, but its application to LLM agent memory in text games is new.

What Already Exists: Memory updating is known in cognitive science and reinforcement learning.

What is Novel: The explicit, context-sensitive updating of both memory types in LLM agents for text games is novel.

References:

  • Schacter (1999) The seven sins of memory [memory updating in humans]
  • Weston et al. (2015) Memory Networks [memory update in neural networks]

New Predictions (Likely outcome)

  1. LLM agents with both episodic and semantic memory modules will outperform those with only one on tasks requiring both recall and generalization.
  2. Agents that adaptively update both memory types in response to novel events will solve dynamic or changing puzzles more efficiently.

New Predictions (Unknown outcome/high-entropy)

  1. If episodic and semantic memory are tightly coupled, agents may develop emergent meta-reasoning strategies not seen in humans.
  2. The optimal balance between episodic and semantic memory usage may depend on the structure of the text game and could vary widely.

Negative Experiments

  1. If agents with only one memory type (episodic or semantic) perform as well as those with both, the theory is challenged.
  2. If adaptive memory updating does not improve performance in dynamic games, the law is called into question.

Unaccounted for Evidence

  1. The impact of memory interference or overload (e.g., too many episodic traces) is not addressed.
