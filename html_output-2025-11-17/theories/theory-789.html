Theory (General Information)

ID: theory-789

Name: Hierarchical Episodic-Semantic Memory Coordination

Type: general

Theory Query: Build a theory of how language model agents can best use memory to solve tasks.

Description: This theory proposes that language model agents achieve superior task performance by coordinating two distinct but interacting memory systems: an episodic memory that stores temporally ordered, context-rich experiences, and a semantic memory that encodes abstracted, generalized knowledge. The agent dynamically decides which memory system to query or update based on the nature of the task, the novelty of the context, and the need for generalization versus specificity. This coordination enables both rapid adaptation to new situations and robust performance on familiar tasks.

Theory (Details)

Theory Statements

Statement 0: Task-Driven Memory System Selection (qualitative)

ConditionDetails
IF
• agent → is_solving → task, and
• task → has_property → novelty_or_specificity_requirement
THEN
• agent → selects_memory_system → episodic_if_novel_or_specific_semantic_if_familiar_or_general

Supporting Evidence for this Law

  1. Cognitive science distinguishes episodic and semantic memory systems, with humans flexibly using each depending on task demands (Tulving, 1972; McClelland et al., 1995).
  2. Recent LLM research shows that agents benefit from both storing specific experiences (episodic) and abstracted knowledge (semantic) (Madotto et al., 2020).
  3. Meta-learning studies indicate that episodic memory is critical for rapid adaptation, while semantic memory supports generalization (Botvinick et al., 2019).

Existing Law Comparison

Likely Classification: somewhat-related-to-existing

Explanation: While the memory systems are known, their dynamic, task-driven coordination in LLM agents is a new synthesis.

What Already Exists: The distinction between episodic and semantic memory is well-established in cognitive science and has been explored in neural architectures.

What is Novel: The explicit, dynamic selection and coordination of these systems in LLM agents based on task novelty and specificity is novel.

Statement 1: Cross-System Memory Consolidation (qualitative)

ConditionDetails
IF
• agent → has_experiences_in → episodic_memory, and
• agent → detects → repeated_patterns_or_generalizable_knowledge
THEN
• agent → consolidates → episodic_experiences_into_semantic_memory

Supporting Evidence for this Law

  1. Cognitive neuroscience shows that humans consolidate episodic experiences into semantic memory over time (McClelland et al., 1995; Kumaran et al., 2016).
  2. Neural network models with memory consolidation mechanisms show improved generalization and reduced catastrophic forgetting (Kirkpatrick et al., 2017).

Existing Law Comparison

Likely Classification: somewhat-related-to-existing

Explanation: The consolidation process is known in biology, but its explicit, dynamic implementation in LLM agents is new.

What Already Exists: Memory consolidation from episodic to semantic is established in neuroscience and some neural models.

What is Novel: The explicit, agent-driven consolidation process in LLM agents, triggered by pattern detection, is novel.

References:

  • McClelland et al. (1995) Why there are complementary learning systems in the hippocampus and neocortex [memory consolidation]
  • Kumaran et al. (2016) What learning systems do intelligent agents need? [episodic-semantic consolidation]
  • Kirkpatrick et al. (2017) Overcoming catastrophic forgetting in neural networks [consolidation in neural models]

New Predictions (Likely outcome)

  1. Agents with coordinated episodic and semantic memory systems will outperform those with only one type of memory on tasks requiring both rapid adaptation and generalization.
  2. Explicit consolidation of repeated episodic experiences into semantic memory will reduce catastrophic forgetting in continual learning settings.

New Predictions (Unknown outcome/high-entropy)

  1. If agents can autonomously discover when to consolidate episodic into semantic memory, they may develop emergent abstraction capabilities.
  2. Dynamic coordination of memory systems may enable agents to transfer knowledge across highly dissimilar tasks.

Negative Experiments

  1. If agents with both memory systems do not outperform those with only one on tasks requiring both specificity and generalization, the theory would be challenged.
  2. If consolidation of episodic into semantic memory does not reduce forgetting or improve generalization, the theory's mechanism would be called into question.

Unaccounted for Evidence

  1. The theory does not specify how to resolve conflicts between episodic and semantic memory when they provide contradictory information.
