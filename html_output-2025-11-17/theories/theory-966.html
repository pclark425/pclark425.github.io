Theory (General Information)

ID: theory-966

Name: Hierarchical Episodic-Semantic Memory Integration Theory

Type: general

Theory Query: Build a theory of how LLM agents for text games can best use memory to solve text game tasks.

Description: This theory posits that LLM agents in text games achieve optimal task performance by integrating hierarchical episodic and semantic memory systems. Episodic memory encodes temporally ordered, context-rich experiences, while semantic memory encodes abstracted, generalized knowledge about the game world. The agent dynamically selects between these memory types based on the current task phase, uncertainty, and novelty, enabling both flexible adaptation to new situations and efficient exploitation of learned regularities.

Theory (Details)

Theory Statements

Statement 0: Hierarchical Memory Selection (qualitative)

ConditionDetails
IF
• LLM agent → is_deciding_action → in text game, and
• task phase → is → exploration or exploitation
THEN
• LLM agent → prioritizes → episodic memory during exploration, semantic memory during exploitation

Supporting Evidence for this Law

  1. Human and animal studies show episodic memory is used for novel situations, semantic for routine tasks.
  2. RL agents with separate episodic and semantic memory modules outperform those with monolithic memory in complex environments.
  3. Text games often require both recalling specific past events and applying general knowledge.

Existing Law Comparison

Likely Classification: somewhat-related-to-existing

What is Novel: Dynamic, phase-dependent prioritization of memory types in LLM agents for text games.

Statement 1: Uncertainty-Guided Memory Integration (qualitative)

ConditionDetails
IF
• LLM agent → detects → high uncertainty or novelty in current state
THEN
• LLM agent → increases_weight → episodic memory retrieval, and
• LLM agent → decreases_weight → semantic memory retrieval

Supporting Evidence for this Law

  1. Humans rely more on episodic memory when facing novel or ambiguous situations.
  2. RL agents with uncertainty-driven memory selection adapt faster to changing environments.
  3. Text games often present unexpected events requiring recall of specific prior experiences.

Existing Law Comparison

Likely Classification: somewhat-related-to-existing

Explanation: The general principle is known, but its application to LLM agent memory integration in text games is new.

What Already Exists: Uncertainty-driven exploration in RL; episodic/semantic memory in cognitive science.

What is Novel: Explicit, uncertainty-guided weighting of memory types in LLM agents for text games.

References:

  • Gershman & Daw (2017) Reinforcement learning and episodic memory in humans and animals [uncertainty and memory]
  • Pritzel et al. (2017) Neural Episodic Control [episodic memory in RL]
  • Madotto et al. (2020) Exploration based language learning for text-based games [exploration/exploitation in text games]

New Predictions (Likely outcome)

  1. LLM agents with hierarchical episodic-semantic memory integration will outperform agents with only one memory type in tasks requiring both recall and generalization.
  2. Agents will shift toward episodic memory retrieval when encountering novel game states or unexpected outcomes.
  3. Semantic memory will dominate in repetitive or well-learned segments of the game.

New Predictions (Unknown outcome/high-entropy)

  1. Agents may develop hybrid memory traces that blend episodic and semantic features.
  2. Over-reliance on one memory type may lead to suboptimal performance in highly dynamic or deceptive games.
  3. Emergent meta-cognitive strategies for memory arbitration may arise.

Negative Experiments

  1. If agents with only episodic or only semantic memory perform as well as those with integrated systems, the theory would be challenged.
  2. If uncertainty-driven memory weighting does not improve adaptation to novel game events, the theory would be called into question.

Unaccounted for Evidence

  1. The theory does not address how to resolve conflicts between episodic and semantic memory outputs.
  2. The impact of memory capacity limits on integration is not specified.
