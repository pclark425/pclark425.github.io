Theory (General Information)

ID: theory-1452

Name: Dynamic Schema Assembly Theory

Type: general

Theory Query: Build a theory of the representational format of conceptual knowledge in brains at a functional (not neural) level.

Description: This theory proposes that conceptual knowledge is functionally represented as dynamic assemblies of schemas—structured templates encoding prototypical features, roles, and relations. Schemas are flexibly composed and decomposed in working memory according to context, allowing for rapid adaptation, generalization, and creative recombination. The representational format is inherently hierarchical, supporting both specific instances and abstract generalizations.

Theory Statements

Statement 0: Schema Assembly Law (qualitative)

ConditionDetails
IF
• conceptual task → is performed → by brain
THEN
• schemas → are assembled → in working memory, and
• schemas → encode → prototypical features, roles, and relations

Supporting Evidence for this Law

  1. Behavioral and neuroimaging studies show rapid schema activation and recombination during conceptual tasks.
  2. Schema-based models explain generalization and transfer across domains.
  3. Lesion studies reveal deficits in schema assembly and flexible reasoning.

Existing Law Comparison

Likely Classification: closely-related-to-existing

Explanation: While schema theory exists, the focus on dynamic assembly in working memory is a novel extension.

What Already Exists: Schema theory is well-established in cognitive psychology.

What is Novel: This law emphasizes dynamic, context-sensitive assembly and decomposition of schemas as a core representational format.

References:

  • Bartlett (1932) Remembering: A Study in Experimental and Social Psychology [schema theory]
  • Ghosh & Gilboa (2014) What is a memory schema? [schema-based memory]

Statement 1: Hierarchical Schema Law (qualitative)

ConditionDetails
IF
• concept → is represented → as schema
THEN
• schema → is hierarchical → with levels for instances and abstractions

Supporting Evidence for this Law

  1. Hierarchical organization of concepts is observed in semantic memory and language.
  2. Neuroimaging shows activation of both specific and abstract schema levels depending on task demands.

Existing Law Comparison

Likely Classification: somewhat-related-to-existing

Explanation: The law extends prior work by making hierarchy a necessary property of the representational format.

What Already Exists: Hierarchical organization of concepts is a classic finding in cognitive science.

What is Novel: This law formalizes hierarchy as a property of the schema-based representational format, not just a property of knowledge organization.

References:

  • Collins & Quillian (1969) Retrieval time from semantic memory [hierarchical semantic networks]
  • Ghosh & Gilboa (2014) What is a memory schema? [schema-based memory]

New Predictions (Likely outcome)

  1. Tasks requiring rapid generalization will show increased neural signatures of schema assembly in working memory regions.
  2. Disruption of working memory will impair flexible schema recombination but not retrieval of static knowledge.

New Predictions (Unknown outcome/high-entropy)

  1. Artificially enhancing schema assembly (e.g., via neurostimulation) will improve creative problem-solving.
  2. Training artificial agents with dynamic, hierarchical schema assembly will yield human-like transfer and abstraction.

Negative Experiments

  1. If conceptual tasks can be performed without any evidence of schema assembly or hierarchy, the theory would be undermined.
  2. If working memory disruption does not impair schema-based generalization, the theory's core claim would be challenged.

Unaccounted for Evidence

  1. The theory does not specify how entirely novel concepts are integrated into existing schema hierarchies.

Existing Theory Comparison

Likely Classification: somewhat-related-to-existing

Explanation: The theory synthesizes and extends prior work, making explicit, testable claims about dynamic schema assembly and hierarchy.
