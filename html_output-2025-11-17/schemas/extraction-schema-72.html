Extraction Schema Details for extraction-schema-72

Extraction Schema (General Information)

Schema ID: extraction-schema-72

Extraction Query: Extract any mentions of LLM-based agents that use memory mechanisms to solve text games, including details about the memory architecture, the text games being solved, and performance comparisons.

Extraction Schema (Details)

Field Name Type Description
agent_name str The name of the LLM-based agent system (e.g., 'ReAct', 'Reflexion', 'DEPS', etc.)
agent_description str A brief description of how the agent works, including its architecture and key components.
base_llm_model str The base LLM model used by the agent (e.g., 'GPT-3.5', 'GPT-4', 'LLaMA-2-7B', 'Claude', etc.)
base_llm_size str The size of the base LLM model in parameters (e.g., '7B', '13B', '175B', etc.). Null if not specified.
text_game_name str The name of the specific text game or benchmark (e.g., 'Jericho', 'TextWorld', 'ALFWorld', 'ScienceWorld', specific game titles like 'Zork', etc.)
text_game_description str A brief description of the text game or benchmark, including the type of tasks and challenges it presents.
uses_memory bool Does the agent use any form of memory mechanism? (true, false, or null for no information)
memory_type str The type/category of memory used (e.g., 'episodic memory', 'semantic memory', 'working memory', 'retrieval-augmented', 'experience replay', 'reflection-based', 'graph-based memory', etc.). Null if no memory or not specified.
memory_architecture str Detailed description of how the memory is structured, what information is stored, how it's organized, and how it's updated during gameplay.
memory_retrieval_mechanism str How does the agent retrieve information from memory? (e.g., 'semantic similarity search', 'recency-based', 'relevance scoring', 'attention mechanism', etc.). Null if not specified.
memory_capacity str What is the capacity or size of the memory? (e.g., 'stores last 10 observations', 'unlimited', '1000 token context window', etc.). Null if not specified.
what_is_stored_in_memory str What specific information is stored in the memory? (e.g., 'past observations and actions', 'successful strategies', 'object locations', 'failed attempts', 'reflections on mistakes', etc.)
performance_with_memory str Performance metrics of the agent WITH memory on the text game (e.g., 'success rate: 75%', 'average score: 45.2', 'completion rate: 60%'). Include units and be specific. Null if not reported.
performance_without_memory str Performance metrics of the agent WITHOUT memory (baseline) on the same text game. Include units and be specific. Null if not reported.
has_ablation_study bool Does the paper include an ablation study comparing performance with and without memory, or comparing different memory configurations? (true, false, or null)
memory_improvement_magnitude str If both with-memory and without-memory performance are reported, what is the improvement? (e.g., '+15% success rate', '2x better score', etc.). Null if cannot be determined.
key_findings_about_memory str What are the key findings or insights about how memory helps (or doesn't help) the agent solve text games? Be specific and concise.
memory_limitations str What limitations or failure modes are reported for the memory mechanism? (e.g., 'struggles with long-term dependencies', 'memory retrieval errors', 'catastrophic forgetting', etc.). Null if not discussed.
comparison_with_other_memory_types str If the paper compares multiple memory approaches, summarize the comparison results (e.g., 'episodic memory outperformed semantic memory by 10%'). Null if no comparison.
