Extraction Schema Details for extraction-schema-47

Extraction Schema (General Information)

Schema ID: extraction-schema-47

Extraction Query: Extract any mentions of experiments, studies, or results related to curriculum learning for compositional tasks, compositional generalization performance, primitive skill training, composition depth effects, and generalization gaps between trained and novel compositions.

Extraction Schema (Details)

Field Name Type Description
study_domain str What domain or environment was studied? (e.g., 'text-based games', 'semantic parsing', 'instruction following', 'interactive fiction', 'commonsense reasoning tasks', etc.)
agent_or_model_name str The name of the agent, model, or system being evaluated.
curriculum_structure str Describe the curriculum structure used, if any. Include details about phases, ordering, complexity progression, etc. (e.g., 'primitives first then compositions', 'random ordering', 'gradual complexity increase', 'no curriculum/end-to-end', etc.)
primitive_training_details str Details about how primitive skills were trained, including whether they were trained separately, what performance level was achieved before composition training, and how many primitives were involved. (null if no information)
composition_depth_range str What range of composition depths were tested? (e.g., '2-3 primitives', '2-5 primitives', 'up to 4 primitives', etc. null if no information)
compositional_diversity_description str Describe the compositional diversity in training data - how many variations, different contexts, structural patterns, etc. were included. (null if no information)
performance_trained_compositions str Performance (accuracy, success rate, etc.) on compositions that were seen during training. Include numerical values with units. (null if no information)
performance_novel_compositions str Performance (accuracy, success rate, etc.) on novel compositions of known primitives that were NOT seen during training. Include numerical values with units. (null if no information)
generalization_gap_measured bool Does the study explicitly measure or report the gap between performance on trained vs. novel compositions? (true/false/null)
generalization_gap_value str If measured, what is the generalization gap (difference between trained and novel composition performance)? Include numerical value. (null if no information)
composition_depth_scaling str What relationship was observed between composition depth and performance or generalization gap? (e.g., 'linear decrease', 'exponential decrease', 'super-linear scaling of difficulty', etc. null if no information)
curriculum_vs_baseline_comparison str If curriculum learning was compared to a baseline (e.g., random ordering, no curriculum, end-to-end training), describe the performance difference. Include numerical comparisons if available. (null if no information)
adaptive_curriculum_used bool Was an adaptive curriculum approach used (where curriculum pacing adjusts based on performance)? (true/false/null)
spurious_correlation_effects str Any findings related to spurious correlations in training data and their effects on generalization. (null if no information)
negative_transfer_observed bool Was negative transfer observed when composition training started before primitives were mastered? (true/false/null)
key_findings_summary str A concise, information-dense summary of the key findings most relevant to understanding how curriculum design, primitive training, compositional diversity, and composition depth affect compositional generalization.
supports_or_challenges_theory str Based on the findings, does this result appear to support, challenge, or provide nuance to the Compositional Generalization Gap Theory? Explain briefly. (e.g., 'supports - shows primitive mastery improves generalization', 'challenges - no benefit from curriculum structure', 'nuances - effect depends on architecture', etc.)
