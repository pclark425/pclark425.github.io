synaptic_rule_description Synaptic strength changes are a biphasic function of spike time difference (Δt) between connected neurons: when the presynaptic event (or sensory-evoked response) reliably precedes the postsynaptic spike within a positive timing window, the connection is potentiated; when the postsynaptic spike precedes the presynaptic event (negative Δt) the connection is depressed. The rule is expressed as a timing-dependent curve (potentiation for Δt in one sign, depression for the other), and in the experiments discussed the net effect on subthreshold responses is biphasic (responses preceding the spike strengthened, those following weakened).

Individual pairing used spikes a few milliseconds before/after stimulus.

experimental_or_model Experimental in vivo whole-cell recordings in anesthetized rats (Pawlak et al. protocol) combined with computational modelling to explain temporal rescaling via input spike-time noise.
multi_timescale_interaction The paper shows an interaction between millisecond-scale spike pairing rules (STDP) and a longer effective timing window in intact cortex: in vitro STDP rules operate over tens of milliseconds (~50 ms), but in vivo repeated pairings produced STDP-consistent changes over ~250 ms. Computational modelling in the cited work indicates that presynaptic spike-time jitter / ongoing network activity (noise) broadens the effective window, so fast spike-timing mechanisms can produce changes manifested over longer millisecond ranges in the intact brain.
key_findings Pawlak and colleagues demonstrated in vivo that repeated pairing of a visual stimulus with an elicited postsynaptic spike changes a neuron's tuning in a manner consistent with STDP: (1) pairing stimulus then spike (spike a few ms after stimulus) trains a naive neuron to respond to that stimulus (potentiation); (2) the same protocol can re-train an already tuned neuron to a new stimulus location; (3) reversing order (spike before stimulus) erases tuning. Subthreshold membrane responses preceding the induced spike were strengthened while those following were weakened (biphasic change). Crucially, the effective timing window for these changes in vivo was ~250 ms (≈5× longer) than typical in vitro STDP windows (≈50 ms), and modelling showed that input spike-time noise in the intact brain can account for this temporal rescaling.
performance_measure Quantitative temporal measures reported: in vitro STDP timing window ≈50 milliseconds (cited); in vivo effective window observed by Pawlak et al. ≈250 milliseconds. Training protocol: visual stimulus presented for 0.5 s paired repeatedly with single elicited spikes separated by a few milliseconds (relative order defines potentiation vs depression). Measured outcomes were shifts in tuning curves and changes in subthreshold membrane-potential responses (qualitative and shown in figures); no numeric learning-rate constants or retention times (hours/days) are given in this insight article.
