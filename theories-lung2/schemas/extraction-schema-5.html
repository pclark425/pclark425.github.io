Extraction Schema Details for extraction-schema-5

Extraction Schema (General Information)

Schema ID: extraction-schema-5

Extraction Query: Extract any mentions of synaptic plasticity rules, brain connectivity patterns, and how they support learning at different temporal scales (e.g., milliseconds to days).

Extraction Schema (Details)

Field Name Type Description
synaptic_rule_name str The name of the synaptic plasticity rule or learning rule (e.g., STDP, Hebbian learning, BCM rule, homeostatic plasticity, short-term plasticity, metaplasticity, etc.)
synaptic_rule_description str A brief but detailed description of how the synaptic plasticity rule works, including key mechanisms and parameters.
temporal_scale str What temporal scale(s) does this synaptic rule operate on? Be specific with timescales (e.g., milliseconds, seconds, minutes, hours, days). Include multiple scales if applicable.
brain_region_circuit str What brain region, circuit, or neural system is being studied? (e.g., hippocampus, cortex, striatum, specific cortical layers, etc.)
connectivity_pattern str What aspects of brain wiring or connectivity are relevant? (e.g., recurrent connections, feedforward architecture, sparse connectivity, specific connection motifs, network topology, etc.)
learning_type str What type of learning is being studied or supported by these mechanisms? (e.g., associative learning, motor learning, working memory, long-term memory consolidation, skill learning, etc.)
experimental_or_model str Is this an experimental study (in vivo, in vitro, etc.) or a computational/theoretical model? Specify the approach.
multi_timescale_interaction str How do multiple temporal scales interact? Describe any mechanisms or findings about how fast and slow processes work together to support learning.
key_findings str What are the key findings or results about how synaptic rules and brain wiring support learning? Be specific and information-dense.
performance_measure str If applicable, what quantitative measures or performance metrics were reported? (e.g., learning rate, memory retention, accuracy, neural response changes, etc. Include numerical values and units if available.)
slow_vs_fast_learning bool Does the paper explicitly compare or contrast slow versus fast learning processes? (true, false, or null for no information)
consolidation_mechanism str Is there mention of memory consolidation or transfer between timescales (e.g., fast initial learning that consolidates into slow long-term memory)? Describe the mechanism if present.
