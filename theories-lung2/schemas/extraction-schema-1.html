Extraction Schema Details for extraction-schema-1

Extraction Schema (General Information)

Schema ID: extraction-schema-1

Extraction Query: Extract any mentions of driver alterations, oncogenic mutations, or genomic changes in lung adenocarcinoma, including both known drivers in RTK/RAS/RAF pathway and novel/rare drivers, especially those found in tumors lacking canonical driver mutations.

Extraction Schema (Details)

Field Name Type Description
driver_alteration str The specific genetic alteration, mutation, fusion, amplification, or other genomic change identified as a potential driver (e.g. 'EGFR L858R', 'KRAS G12C', 'ALK fusion', 'NF1 loss', etc.)
alteration_type str The type of alteration (e.g. 'point mutation', 'gene fusion', 'amplification', 'deletion', 'loss of function', 'copy number alteration', 'epigenetic change', etc.)
gene_or_pathway str The gene(s) or pathway involved in the alteration
is_rtk_ras_raf_pathway bool Is this alteration in the canonical RTK/RAS/RAF pathway? (true for known drivers like EGFR, KRAS, ALK, ROS1, BRAF, etc.; false for alterations outside this pathway; null if unclear)
frequency_in_luad str What is the frequency/prevalence of this alteration in lung adenocarcinoma? (e.g. '15%', '3-5%', 'rare <1%', etc. Include exact numbers if available, null if not reported)
frequency_in_driver_negative str What is the frequency of this alteration specifically in driver-negative or pan-negative lung adenocarcinomas (i.e., those lacking known RTK/RAS/RAF drivers)? (numerical with units, null if not reported)
evidence_type str What type of evidence supports this being a driver? (e.g. 'genomic profiling', 'functional studies', 'response to targeted therapy', 'mouse models', 'co-occurrence/mutual exclusivity analysis', etc.)
functional_evidence str Describe any functional or mechanistic evidence that this alteration drives tumor growth (e.g. 'increases cell proliferation in vitro', 'tumorigenic in mouse models', 'activates MAPK signaling', etc. null if not reported)
clinical_associations str Any clinical or pathological features associated with this driver (e.g. 'found in never-smokers', 'associated with solid subtype', 'more common in smokers', 'co-occurs with TP53 mutations', etc. null if not reported)
therapeutic_implications str Are there therapeutic implications or targeted therapies for this alteration? (e.g. 'responsive to drug X', 'potential target for Y inhibitor', etc. null if not reported)
study_cohort_size str How many lung adenocarcinoma samples were analyzed in the study? (numerical, null if not reported)
detection_method str What methods were used to detect this alteration? (e.g. 'whole exome sequencing', 'RNA-seq', 'targeted panel sequencing', 'immunohistochemistry', etc.)
novel_or_rare bool Is this described as a novel, rare, or previously uncharacterized driver in lung adenocarcinoma? (true if novel/rare/emerging, false if well-established, null if unclear)
