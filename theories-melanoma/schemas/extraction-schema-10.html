Extraction Schema Details for extraction-schema-10

Extraction Schema (General Information)

Schema ID: extraction-schema-10

Extraction Query: Extract any mentions of combination therapy strategies evaluated to overcome late/acquired/secondary resistance to immune checkpoint inhibitors in melanoma. Capture clinical and preclinical evidence, patient/model context, prior immunotherapy exposure and resistance definitions, mechanistic rationale and targeted resistance mechanisms, biomarkers and their associations, treatment components and schedules, outcomes (responses, survival, intracranial control), safety, sequencing/rechallenge details, and notable limitations.

Extraction Schema (Details)

Field Name Type Description
paper_title str Full title of the paper.
paper_year int Year of publication (numeric if available).
study_type str Type of study: preclinical_in_vitro, preclinical_in_vivo, translational, case_report, retrospective_cohort, prospective_cohort, phase1, phase1b/2, phase2, phase3, meta_analysis, review (extract primary data if present).
melanoma_subtype str Melanoma subtype studied: cutaneous, mucosal, acral, uveal, mixed/unspecified.
disease_setting str Clinical context: metastatic/advanced, adjuvant, neoadjuvant, brain metastases cohort, oligoprogression, other.
resistance_type str Primary vs acquired/secondary vs late resistance; record exact term used.
resistance_definition str How the study defined acquired/late resistance (e.g., progression after initial CR/PR/SD lasting ≥6 months; iRECIST criteria; time thresholds).
prior_immunotherapy str Details of prior ICI(s): agents (e.g., anti–PD-1, anti–PD-L1, anti–CTLA-4), combinations, lines of therapy.
best_prior_response str Best response achieved before resistance (CR, PR, SD, duration if reported).
time_on_prior_immunotherapy str Exposure duration to prior ICI before progression (include units).
time_to_progression_on_io str Time from ICI start (or response) to progression indicating acquired resistance (include units).
BRAF_NRAS_status str Tumor mutation status relevant to targeted therapy (e.g., BRAF V600E/K, NRAS, NF1, wild-type).
baseline_brain_metastases bool Whether patients had brain metastases at baseline (true/false or null).
number_of_prior_lines str Number of prior systemic therapy lines (numeric if available).
combination_strategy_name str Short name of the combination strategy (e.g., nivolumab+relatlimab; ipilimumab+nivolumab salvage; TIL therapy (lifileucel) after PD-1; FMT+pembrolizumab).
components_and_classes str List all components with class/target (e.g., nivolumab [PD-1], ipilimumab [CTLA-4], relatlimab [LAG-3], T-VEC [oncolytic virus], radiotherapy 30 Gy/10 fx).
dosing_schedule str Doses, schedules, routes for each component; include sequencing (concurrent vs sequential) if embedded in dosing.
sequencing_timing str Therapy sequence relative to prior ICI: immediate salvage, delayed rechallenge, switch to targeted then reintroduce ICI, maintenance strategies; intervals/time gaps.
local_therapy_context str Use of local treatments with systemic therapy (e.g., stereotactic radiosurgery for oligoprogression while continuing PD-1; intratumoral injections).
rationale_mechanism str Mechanistic rationale for the combo (e.g., augment priming via CTLA-4, reverse T cell exhaustion via LAG-3, increase antigenicity via epigenetic drugs, inflame tumor via oncolytic/STING).
resistance_mechanism_addressed str Specific resistance mechanisms targeted/observed: B2M/HLA loss, JAK1/2 IFN signaling defects, antigen loss/dedifferentiation (MITF-low/AXL-high), WNT/β-catenin, PTEN loss/PI3K, myeloid immunosuppression (MDSC/TAM), Treg dominance, adenosine/CD73, IDO, VEGF-mediated exclusion, microbiome-related nonresponsiveness.
biomarkers_measured str List of biomarkers assessed: PD-L1, TMB, gene signatures (IFN-γ), HLA class I, B2M/JAK mutations, ctDNA, TCR clonality, microbiome taxa, spatial immune profiling, LDH, AXL/MITF states, others.
biomarker_associations str Associations between biomarkers/resistance mechanisms and outcomes to the combination (e.g., B2M loss predicts poor ICI rechallenge; microbiome composition predicts response to FMT+PD-1).
outcomes_summary str Brief, dense summary of efficacy outcomes and key numbers for the combo in the resistant population.
ORR str Objective response rate with units/percent and CI if available; specify if in PD-1-refractory subset.
CR_rate str Complete response rate (percent, CI if available).
DCR str Disease control rate (percent).
PFS str Progression-free survival (median, months, HR/CI if comparative).
OS str Overall survival (median, months, HR/CI if comparative).
DoR str Duration of response (median, months) if reported.
intracranial_outcomes str Outcomes for brain metastases if reported (intracranial ORR, PFS, local control).
response_by_biomarker_subgroup str Efficacy stratified by biomarkers/resistance mechanism (e.g., PD-L1+, BRAF-mutant, presence of B2M mutation).
comparator_and_results str If a comparator arm exists (e.g., ipi alone vs ipi+nivo salvage), summarize comparator and relative results (HRs, p-values).
rechallenge_specifics_and_results str Details and outcomes for ICI rechallenge strategies (e.g., PD-1 reinitiation +/- CTLA-4, interval off therapy).
safety_grade3plus_AEs str Rates and types of grade ≥3 treatment-related adverse events.
treatment_related_mortality str Number/percent of treatment-related deaths, if any.
discontinuation_rate str Rate of treatment discontinuation due to AEs.
notable_toxicities str Specific toxicities of note (e.g., hepatitis, colitis, cytokine-mediated toxicities, neurotoxicity, autoimmune events).
sample_size str Number of patients/animals analyzed for the resistant cohort; include subgroup sizes where relevant.
trial_name_or_NCT str Trial identifier (e.g., NCT number) or study acronym/name if applicable.
preclinical_model str For preclinical studies: model system (e.g., B16F10 murine melanoma, PDX, syngeneic models), genetic alterations, and resistance modeling approach.
preclinical_results str Key preclinical efficacy results (tumor growth inhibition/regression, survival benefit, synergy metrics) with numbers.
immune_correlates str Changes in TME/immune contexture with the combo (e.g., CD8+ T cell infiltration, Treg/MDSC shifts, cytokines, IFN signaling, antigen presentation).
limitations str Study limitations relevant to inference (small sample, single-arm, selection bias, heterogeneous definitions of resistance, negative/neutral results).
evidence_level str Evidence strength category (e.g., high: randomized phase 3; moderate: phase 2/robust translational; low: retrospective/case series; preclinical).
key_quote str A concise verbatim sentence from the paper that encapsulates the main finding about overcoming acquired resistance.
notes str Any additional details aiding theory building (e.g., context on microbiome donors in FMT, epigenetic drug class specifics, radiotherapy dose/fractionation, site-specific effects).
