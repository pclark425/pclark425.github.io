Extraction Schema Details for extraction-schema-6

Extraction Schema (General Information)

Schema ID: extraction-schema-6

Extraction Query: Extract any mentions of evolutionary differences in brain cell types between humans and other mammalian species, including what cell types differ, how they differ, which species are compared, what brain regions are involved, and proposed evolutionary or functional explanations for these differences.

Extraction Schema (Details)

Field Name Type Description
cell_type_name str The specific name or category of the brain cell type being discussed (e.g. 'von Economo neurons', 'chandelier cells', 'rosehip neurons', 'layer 2/3 pyramidal neurons', 'microglia', etc.)
cell_type_description str A brief description of this cell type, including key characteristics like morphology, markers, or electrophysiological properties.
brain_region str The specific brain region(s) or area(s) where this cell type is found or where differences are observed (e.g. 'prefrontal cortex', 'anterior cingulate cortex', 'hippocampus', 'neocortex layer 2/3', etc.)
human_species str Confirm this involves humans (Homo sapiens). Value should be 'Homo sapiens' or 'humans' if humans are involved, null otherwise.
comparison_species str What other mammalian species are being compared to humans? List all species mentioned (e.g. 'macaque', 'mouse', 'chimpanzee', 'gorilla', 'marmoset', etc.). Be specific with species names.
nature_of_difference str How does this cell type differ between humans and other mammals? Be specific about the nature of the difference (e.g. 'present in humans but absent in other species', '2x higher density in humans', 'different morphology with longer dendrites', 'different gene expression pattern', 'different connectivity pattern', etc.)
quantitative_difference str If quantitative differences are reported (e.g. cell counts, densities, proportions, sizes), extract the specific numbers with units for both humans and comparison species. Null if not reported.
evolutionary_explanation str What evolutionary explanation or hypothesis is proposed for why this difference exists? What selective pressures or evolutionary processes are suggested (e.g. 'expanded social cognition', 'language development', 'extended lifespan', 'larger brain size scaling', 'recent evolutionary innovation', etc.)? Null if not discussed.
functional_purpose str What is the proposed functional role or purpose of this cell type or the difference observed? What does it do, and why does that matter for human capabilities (e.g. 'involved in self-awareness and social cognition', 'enables complex language processing', 'supports working memory', 'inhibits specific cortical outputs', etc.)? Null if not discussed.
evidence_type str What type of evidence or methodology was used to establish this difference? (e.g. 'single-cell RNA sequencing', 'histological analysis', 'immunohistochemistry', 'electron microscopy', 'comparative genomics', 'electrophysiology', 'computational modeling', etc.)
human_specific_or_enriched bool Is this cell type described as human-specific (only found in humans) or human-enriched (present in other species but much more abundant/prominent in humans)? True for human-specific, false for present across species but different, null if unclear.
when_evolved str If discussed, when in evolutionary history did this difference emerge? (e.g. 'after human-chimpanzee split', 'in great apes', 'in primates', 'recent human evolution', 'ancient mammalian feature', etc.). Null if not discussed.
related_to_human_cognition bool Is this cell type or difference explicitly linked to uniquely human or enhanced human cognitive abilities? True if yes, false if no or other functions, null if not discussed.
