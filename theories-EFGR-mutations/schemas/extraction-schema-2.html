Extraction Schema Details for extraction-schema-2

Extraction Schema (General Information)

Schema ID: extraction-schema-2

Extraction Query: Extract any mentions of EGFR mutation frequencies in different ethnic or geographic populations, particularly comparing East Asian and non-Asian populations, along with any proposed mechanisms, risk factors, or explanations for these differences.

Extraction Schema (Details)

Field Name Type Description
study_population str The ethnic/geographic population(s) studied (e.g., 'East Asian', 'Caucasian', 'Chinese', 'Japanese', 'European', 'North American', etc.). Be specific.
egfr_mutation_frequency str The frequency/prevalence of EGFR mutations in the population. Include specific percentages and which exons if mentioned (e.g., '50% overall, 30% exon 19 deletions, 15% exon 21 L858R'). null if no information.
comparison_populations str If the study compares multiple populations, list them and their respective EGFR mutation frequencies (e.g., 'Asian: 45%, Caucasian: 15%'). null if no comparison.
specific_mutation_types str Specific EGFR mutation types discussed (e.g., 'exon 19 deletions', 'L858R in exon 21', 'T790M', etc.). null if not specified.
smoking_status_association str Is there information about smoking status and its relationship to EGFR mutations in different populations? (e.g., 'EGFR mutations more common in never-smokers, especially in Asian females'). null if no information.
environmental_factors str Any environmental factors mentioned that might explain ethnic differences (e.g., 'air pollution', 'cooking oil fumes', 'specific carcinogen exposures', 'radon', etc.). null if none mentioned.
genetic_factors str Any genetic or genomic factors mentioned that might explain ethnic differences (e.g., 'germline polymorphisms', 'DNA repair gene variants', 'specific SNPs associated with ethnicity', etc.). null if none mentioned.
biological_mechanisms str Any biological or molecular mechanisms proposed to explain why EGFR mutations are more common in certain populations. Be detailed and specific.
lung_cancer_subtype str Lung cancer histological subtype(s) discussed (e.g., 'adenocarcinoma', 'squamous cell carcinoma', 'non-small cell lung cancer'). null if not specified.
demographic_factors str Other demographic factors that differ between populations (e.g., 'age', 'gender distribution', 'never-smoker prevalence'). null if none mentioned.
proposed_explanation str What explanation(s) does the paper propose for the ethnic/geographic differences in EGFR mutation frequency? Be comprehensive and include multiple explanations if given.
counterevidence_or_confounds str Any counter-evidence, alternative explanations, or confounding factors mentioned that complicate the ethnic difference pattern. null if none mentioned.
