Paper Title: Accurate diagnosis of dementia and Alzheimer's with deep network approach based on multi‐channel feature extraction and selection

Paper Abstract: In this article, we have proposed a multi‐stage in‐depth approach based on the improved VGGNet architecture for automatically and accurately diagnosing dementia and Alzheimer's disease. In this approach, first of all, the learned weights of the VGG16 architecture are frozen, and multichannel attributes are extracted from each pooling layer. Then, these attributes were given to the inputs of the attribute average pooling layers, and one‐dimensional attributes were produced using the flattened layer. Distinctive and effective attributes were selected from these deep attributes by the mRMR algorithm. Finally, the selected attributes are given to the input of the eight‐layer classification model, which includes the Fully connected, Relu, and softmax layers. A publicly available data set consisting of four classes and 6400 images was used to test the correctness of the proposed architecture. In addition, since the number of images belonging to the classes in this data set is unstable, data augmentation methods were used. As a result, a 98.6% accuracy score was produced with the developed architecture. These results show that the proposed architecture outperforms the original VGG16 and previous works.

| INTRODUCTION

| MATERIALS AND METHODS

In the study, we proposed a new deep approach based on improved VGGNET architecture to detect Alzheimer's disease.The general representation of the proposed approach improved based on pre-trained VGGNet architecture is shown in Figure 1.

As can be seen in Figure 1, the proposed model consists of three main stages.In the first stage, feature maps of size M Â N Â C were obtained from the improved VGG16 architecture through pooling layers.Then, these feature maps were reduced to 1 Â C in size using feature average pooling.In the second stage, the features obtained from five feature average pooling layers were combined.A feature selection algorithm (MRMR) was used to select high-performance and effective features from these combined features.In the final stage, the selected features were given to the input of an 8-layer neural network classifier.In this way, the proposed model was designed to successfully perform the image classification task following a complex architecture.More details on the components of the proposed model are given in the sub-sections.

| Feature selection

The Minimum Redundancy Maximum Relevance, called MRMR, is an algorithm for selecting a subset of features that are most relevant to a given classification task while minimizing the redundancy among the selected features. 34It is based on the concept of mutual information and uses a greedy search strategy to identify the best subset of features.][36] The MRMR algorithm starts by initializing an empty feature set and iteratively adding one feature at a time.For each feature, the algorithm calculates two scores: the mutual information (MI) between the feature and the class label and the minimum redundancy (MR) between the feature and the features already in the feature set.The feature with the highest MRMR score (MI-MR) is then added to the feature set.The process is repeated until a pre-specified number of features is reached or until no feature has a positive MRMR score. 34,35,37,38he mathematical formula for the mutual information (MI) between feature M and class label N is 34 : MI M, N ð Þ¼sum p a, b ð ÞÂlog p a, b ð Þ= p a ð ÞÂp b ð Þ ð Þ ð Þ ð ð6Þ The mutual information score measures the degree of dependency between the feature and the class label and is used to quantify the relevance of the feature to the classification task.In Equation ( 6), p(a) and p(b) are the marginal probability distributions of M and N, respectively, and p(a,b) is the joint probability distribution of M and N. | CONCLUSIONS
