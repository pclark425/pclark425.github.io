Extraction Schema Details for extraction-schema-117

Extraction Schema (General Information)

Schema ID: extraction-schema-117

Extraction Query: Extract any mentions of large language models (LLMs) being used to estimate or predict the probability or likelihood of specific future real-world scientific discoveries, including details on methods, evaluation, results, and limitations.

Extraction Schema (Details)

Field Name Type Description
model_name str The name of the LLM model used for probability estimation or forecasting (e.g., GPT-4, PaLM, Llama-2).
model_description str A brief description of the LLM model, including architecture, size, and any relevant training details.
model_size str The size of the model in parameters (e.g., 7B, 13B, 70B, etc.), or null if not specified.
prediction_target str A description of the specific scientific discovery or event whose probability is being estimated (e.g., discovery of a new material, breakthrough in physics, approval of a new drug, etc.).
prediction_method str A description of the method or approach used by the LLM to estimate or predict the probability/likelihood (e.g., prompt engineering, chain-of-thought, calibration techniques, etc.).
evaluation_protocol str How the accuracy or calibration of the LLM's probability estimates is evaluated (e.g., retrospective analysis, comparison to human experts, Brier score, log-likelihood, etc.).
dataset_or_benchmark str The name and brief description of any dataset, benchmark, or set of scientific events used for evaluation.
results str A concise summary of the results, including quantitative performance metrics if available (e.g., accuracy, calibration error, comparison to baselines or humans).
limitations_or_challenges str Any reported limitations, challenges, or failure cases in using LLMs for probability estimation of scientific discoveries.
