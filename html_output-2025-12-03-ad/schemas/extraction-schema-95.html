Extraction Schema Details for extraction-schema-95

Extraction Schema (General Information)

Schema ID: extraction-schema-95

Extraction Query: Extract any mentions of selective vulnerability or resistance of specific neuronal populations, cell types, layers, or regions within the visual cortex to amyloid-β (amyloid-beta) plaques and tau protein tangles in the context of dementia or Alzheimer's disease.

Extraction Schema (Details)

Field Name Type Description
neuronal_population_type str The specific type of neuronal population, cell type, cortical layer, or visual cortex region being studied (e.g., 'layer 5 pyramidal neurons', 'parvalbumin-positive interneurons', 'V1 primary visual cortex', 'MT/V5 motion area', etc.)
vulnerability_status str Is this neuronal population vulnerable, resistant, or intermediate to amyloid-β and/or tau pathology? Be specific about which pathology if only one is mentioned.
amyloid_beta_measurement str Quantitative or qualitative measurements of amyloid-β plaque burden, density, or accumulation in this neuronal population (include units and numerical values if available, null if no information)
tau_measurement str Quantitative or qualitative measurements of tau tangle burden, density, or accumulation in this neuronal population (include units and numerical values if available, null if no information)
cellular_properties str Specific cellular or molecular properties of this neuronal population that are mentioned in relation to vulnerability or resistance (e.g., metabolic rate, calcium buffering capacity, connectivity patterns, gene expression profiles, protein levels, etc.)
proposed_mechanism str The proposed mechanism or explanation for why this neuronal population is vulnerable or resistant to pathology. Be detailed and specific.
comparative_population str If the study compares this population to another population with different vulnerability, name the comparison population and briefly describe the key differences.
species_and_model str What species and experimental model was used? (e.g., 'human post-mortem tissue', 'APP/PS1 mouse model', '3xTg-AD mice', etc.)
dementia_stage str What stage of dementia or disease progression was studied? (e.g., 'early-stage AD', 'mild cognitive impairment', 'advanced AD', 'preclinical', null if no information)
key_findings str A concise, information-dense summary of the key findings related to selective vulnerability in the visual cortex. Focus on results that would help explain WHY certain populations are vulnerable while others are resistant.
