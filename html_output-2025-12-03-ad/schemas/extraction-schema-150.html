Extraction Schema Details for extraction-schema-150

Extraction Schema (General Information)

Schema ID: extraction-schema-150

Extraction Query: Extract any mentions of language model agents using memory to solve tasks, including details of the memory mechanism, tasks, comparative results, ablations, and key findings.

Extraction Schema (Details)

Field Name Type Description
agent_name str The name of the language model agent being evaluated (e.g. 'ReAct', 'AutoGPT', 'Toolformer', etc.).
agent_description str A brief description of the agent, including its architecture or unique features.
model_name str The name of the underlying language model (e.g. 'GPT-4', 'Llama-2-70B', etc.), if specified.
model_description str A brief description of the underlying language model, including size or notable properties.
task_name str The name of the task or benchmark the agent is evaluated on (e.g. 'HotpotQA', 'WebShop', 'ALFWorld').
task_description str A brief description of the task, including what the agent is required to do.
task_type str The type or category of the task (e.g. 'question answering', 'web navigation', 'multi-step reasoning', etc.).
memory_used bool Does the agent use any form of memory to solve the task? (true, false, or null if not specified)
memory_type str The type of memory used (e.g. 'episodic', 'working', 'external database', 'retrieval-augmented', 'scratchpad', etc.).
memory_mechanism str A description of how the memory is implemented or accessed (e.g. 'retrieval from vector database', 'attention over previous steps', 'explicit memory buffer', etc.).
memory_representation str What is stored in memory? (e.g. 'previous actions', 'intermediate thoughts', 'external documents', 'API call results', etc.)
memory_retrieval_method str How does the agent retrieve information from memory? (e.g. 'semantic search', 'recency-based', 'prompt concatenation', etc.)
performance_with_memory str The performance of the agent on the task when using memory (numerical, include units and metrics; null if not reported).
performance_without_memory str The performance of the agent on the task without using memory (numerical, include units and metrics; null if not reported).
has_performance_with_without_memory bool Does the paper report performance both with and without memory for this agent/task? (true, false, or null if not specified)
ablation_or_comparison str Any ablation studies, comparisons, or analysis results regarding memory usage (e.g. 'removing memory reduces performance by 20%', 'retrieval-augmented memory outperforms simple buffer').
key_findings str Concise summary of the main findings or recommendations about memory use in language model agents from the paper.
limitations_or_challenges str Any reported limitations, challenges, or negative results related to memory use in language model agents.
