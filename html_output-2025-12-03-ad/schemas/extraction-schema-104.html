Extraction Schema (General Information)

Schema ID: extraction-schema-104

Extraction Query: Extract any mentions of LLM-based agents playing text games, with a focus on how memory is used, the type of memory mechanisms, comparative performance with and without memory, and any recommendations or challenges regarding memory usage.

Extraction Schema (Details)

Field Name Type Description
agent_name str The name of the LLM-based agent or model evaluated in the paper (e.g. 'GPT-3 agent', 'LLM-TextWorld Agent').
agent_description str A brief description of the agent, including its architecture and how it uses an LLM.
llm_model_name str The name of the underlying LLM used (e.g. GPT-3, GPT-4, PaLM, Llama-2, etc.).
game_or_benchmark_name str The name of the text game, interactive fiction, or benchmark used for evaluation (e.g. 'TextWorld', 'Jericho', 'Zork').
task_description str A brief description of the specific task(s) or objectives the agent is evaluated on within the text game.
memory_used bool Does the agent use any form of memory mechanism beyond the LLM's context window? (true, false, or null if not specified)
memory_type str A description of the memory mechanism used (e.g. episodic memory, external memory, retrieval-augmented, scratchpad, summarization, key-value store, etc.).
memory_representation str What is stored in memory? (e.g. raw text, structured facts, action history, summarized states, embeddings, etc.)
memory_update_mechanism str How is the memory updated during gameplay? (e.g. after each action, periodically, on demand, etc.)
memory_retrieval_mechanism str How does the agent retrieve or use information from memory? (e.g. attention, search, retrieval-augmented generation, etc.)
performance_with_memory str Performance of the agent on the task/game when using memory (include metrics, units, and any relevant details; null if not reported).
performance_without_memory str Performance of the agent on the task/game without using memory (include metrics, units, and any relevant details; null if not reported).
has_performance_comparison bool Does the paper report a direct comparison of performance with and without memory? (true, false, or null if not specified)
ablation_or_analysis str Summary of any ablation studies, analyses, or experiments comparing different memory strategies or configurations.
challenges_or_limitations str Any reported challenges, limitations, or failure cases related to memory usage in LLM agents for text games.
best_practices_or_recommendations str Any explicit recommendations, conclusions, or best practices for using memory in LLM agents for text games, as stated by the authors.
