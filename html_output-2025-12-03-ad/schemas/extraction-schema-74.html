Extraction Schema Details for extraction-schema-74

Extraction Schema (General Information)

Schema ID: extraction-schema-74

Extraction Query: Extract any mentions of language model agents using memory to solve tasks, including details of the agent, the memory mechanism, the tasks, and performance comparisons.

Extraction Schema (Details)

Field Name Type Description
agent_name str The name of the language model agent being described (e.g. 'ReAct', 'AutoGPT', 'MemGPT', etc.).
agent_description str A brief description of the agent, including its architecture and main features.
memory_used bool Does the agent use a memory mechanism beyond the standard LLM context window? (true, false, or null if not specified)
memory_type str Type of memory used by the agent (e.g. 'retrieval-augmented', 'episodic memory', 'external database', 'scratchpad', 'context window extension', etc.).
memory_mechanism_description str A concise description of how the memory is implemented and used (e.g. how information is stored, retrieved, updated, and integrated with the agent's reasoning).
task_name str The name of the task or benchmark the agent is evaluated on (e.g. 'HotpotQA', 'WebShop', 'ALFWorld', 'Toolformer', etc.).
task_description str A brief description of the task, including its main challenge (e.g. multi-step reasoning, tool use, long-horizon planning, dialogue, etc.).
task_type str The general type of task (e.g. 'question answering', 'planning', 'tool use', 'dialogue', 'game playing', etc.).
performance_with_memory str Performance of the agent on the task when using memory (numerical, include units and metrics; null if not reported).
performance_without_memory str Performance of the agent on the task without using memory (numerical, include units and metrics; null if not reported).
has_performance_comparison bool Does the paper report a direct comparison of performance with and without memory? (true, false, or null if not specified)
key_findings str A concise summary of the key findings regarding the use of memory in the agent (e.g. 'episodic memory improves long-horizon task performance by 20%', 'retrieval-augmented memory helps with factual recall', etc.).
limitations_or_challenges str Any reported limitations, challenges, or failure cases related to the use of memory in the agent (e.g. 'memory retrieval errors', 'scaling issues', 'catastrophic forgetting', etc.; null if not reported).
