Extraction Schema Details for extraction-schema-23

Extraction Schema (General Information)

Schema ID: extraction-schema-23

Extraction Query: Extract any mentions of systems or methods that use probabilistic symbolic world models (such as PPDDL, PDDL, or belief states) for planning in text-based environments, especially those that integrate uncertainty from large language models.

Extraction Schema (Details)

Field Name Type Description
system_name str The name of the system, model, or method that uses symbolic world models for text environments.
system_description str A detailed description of the system, including its key components and how it works.
world_model_type str What type of symbolic world model is used? (e.g., PPDDL, PDDL, lifted belief states, probabilistic state machines, logic-based representations, etc.)
world_model_description str A detailed description of how the world model represents states, actions, and transitions. Include whether it's probabilistic or deterministic.
uses_llm bool Does the system use a large language model (LLM) as a component? (true, false, or null for no information)
llm_role str If an LLM is used, what role does it play? (e.g., action generation, state estimation, world model construction, reward prediction, etc.)
llm_model_name str If an LLM is used, what is the name/type of the LLM? (e.g., GPT-3, GPT-4, BERT, T5, etc.)
uncertainty_modeling bool Does the system explicitly model uncertainty in the world model or LLM outputs? (true, false, or null for no information)
uncertainty_type str If uncertainty is modeled, what type of uncertainty is captured? (e.g., epistemic uncertainty, aleatoric uncertainty, LLM prediction uncertainty, state uncertainty, action outcome uncertainty, etc.)
uncertainty_method str If uncertainty is modeled, what method is used to represent and reason about it? (e.g., probability distributions, belief states, Monte Carlo sampling, ensemble methods, confidence scores, etc.)
planning_algorithm str What planning algorithm or method is used? (e.g., MCTS, A*, value iteration, policy gradient, hierarchical planning, etc.)
planning_integrates_uncertainty bool Does the planning algorithm explicitly integrate uncertainty into its decision-making? (true, false, or null for no information)
text_environment_name str What text-based environment or benchmark is used for evaluation? (e.g., TextWorld, Jericho, ALFWorld, ScienceWorld, specific interactive fiction games, etc.)
text_environment_description str A brief description of the text environment, including its complexity and key characteristics.
performance_metric str What metrics are used to evaluate performance? (e.g., success rate, average reward, completion rate, number of steps, etc.)
performance_value str What is the performance of the system on the text environment? Include numerical values with units. Be specific about conditions (e.g., 'with uncertainty modeling: 85% success rate').
baseline_comparison str What baseline methods are compared against, and what are their performance values? (e.g., 'non-symbolic LLM agent: 60% success rate', 'deterministic world model: 70% success rate')
has_ablation_uncertainty bool Does the paper report ablation results comparing performance with and without uncertainty modeling? (true, false, or null for no information)
ablation_results str If ablation studies are performed, what are the key findings? Especially regarding the impact of uncertainty modeling, symbolic representations, or LLM integration.
key_findings str What are the key findings or insights from this work regarding probabilistic symbolic world models for text environments with LLM uncertainty?
