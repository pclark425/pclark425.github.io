Extraction Schema Details for extraction-schema-112

Extraction Schema (General Information)

Schema ID: extraction-schema-112

Extraction Query: Extract any mentions of LLMs being evaluated on cognitive psychology tests, including details of the tests, LLM performance, human baseline performance, and any direct comparisons or notable differences.

Extraction Schema (Details)

Field Name Type Description
model_name str The name of the LLM model being evaluated (e.g., GPT-4, Llama 2, PaLM, etc.).
model_description str A brief description of the LLM model, including architecture, training data, or other relevant details.
model_size str The size of the LLM model, typically in number of parameters (e.g., 7B, 13B, 70B, etc.).
cognitive_test_name str The name of the cognitive psychology test or battery (e.g., Raven's Progressive Matrices, Stroop Test, Digit Span, etc.).
cognitive_test_type str The type or category of the cognitive test (e.g., memory, reasoning, attention, executive function, language, etc.).
cognitive_test_description str A brief description of the cognitive test, including what it measures and how it is administered.
llm_performance str The performance of the LLM on the cognitive test, including quantitative results (e.g., accuracy, score, percent correct) and qualitative observations if available.
human_baseline_performance str The performance of normal (neurotypical) humans on the same cognitive test, including quantitative results and relevant demographic details if available.
performance_comparison str A direct comparison between LLM and human performance on the test, including whether the LLM outperforms, matches, or underperforms relative to humans, and any statistical significance if reported.
notable_differences_or_limitations str Any notable differences, limitations, or caveats in the LLM's performance compared to humans, such as failure modes, qualitative differences, or test-specific challenges.
