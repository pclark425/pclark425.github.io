Extraction Schema Details for extraction-schema-33

Extraction Schema (General Information)

Schema ID: extraction-schema-33

Extraction Query: Extract any mentions of models or agents that are pretrained on text-based environments or language data and then transferred to 3D embodied tasks, including details about the pretraining, the embodied tasks, transfer performance, action mappings, and sample complexity.

Extraction Schema (Details)

Field Name Type Description
model_agent_name str The name of the model or agent that was pretrained on text and transferred to embodied tasks.
model_agent_description str A brief description of the model or agent architecture and key characteristics.
pretraining_data_type str What type of text-based data or text world was used for pretraining? (e.g. 'text-based games', 'language instructions', 'Wikipedia', 'interactive fiction', 'procedurally generated text environments', etc.)
pretraining_data_details str Detailed description of the pretraining data, including dataset names, size, and characteristics.
embodied_task_name str The name of the 3D embodied task or benchmark that the model was transferred to.
embodied_task_description str A detailed description of the 3D embodied task, including the environment, objectives, and requirements.
action_space_text str Description of the action space in the text pretraining environment (e.g. 'high-level commands like go north, take key', 'natural language instructions', etc.)
action_space_embodied str Description of the action space in the 3D embodied environment (e.g. 'continuous motor controls', 'discrete navigation actions', 'joint angles', etc.)
action_mapping_method str How were high-level action semantics from text mapped to low-level actions in the embodied environment? Describe the mapping method or architecture used.
perception_requirements str What perception modalities and capabilities were required in the embodied task? (e.g. 'RGB vision', 'depth sensing', 'object detection', etc.)
transfer_successful bool Was transfer from text pretraining to embodied tasks successful? (true if positive transfer occurred, false if negative transfer or no benefit, null if unclear)
performance_with_pretraining str Performance on the embodied task when using text pretraining. Include numerical results with units if available.
performance_without_pretraining str Performance on the embodied task without text pretraining (baseline). Include numerical results with units if available.
sample_complexity_with_pretraining str How many samples/episodes were needed to reach a performance threshold with pretraining? Include numerical results if available.
sample_complexity_without_pretraining str How many samples/episodes were needed to reach a performance threshold without pretraining? Include numerical results if available.
sample_complexity_gain str What was the gain in sample efficiency from pretraining? (e.g. '10x fewer samples needed', '50% reduction in training time', etc.)
transfer_success_factors str What factors contributed to successful transfer? (e.g. 'semantic similarity between text and embodied tasks', 'shared action vocabulary', 'object-centric representations', etc.)
transfer_failure_factors str What factors limited or prevented successful transfer? (e.g. 'perception gap', 'action space mismatch', 'lack of spatial reasoning in text', etc.)
key_findings str Summarize the key findings about when and how text pretraining transfers to embodied tasks in a concise, information-dense way.
