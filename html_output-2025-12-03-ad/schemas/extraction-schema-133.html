Extraction Schema (General Information)

Schema ID: extraction-schema-133

Extraction Query: Extract any mentions of the representational format of conceptual knowledge in the brain at a functional level, including theoretical models, their descriptions, claimed representational formats, supporting or contradictory empirical evidence, experimental paradigms, key findings, and citation information.

Extraction Schema (Details)

Field Name Type Description
theory_name str The name of the theory or model of conceptual knowledge representation (e.g., "Prototype Theory", "Exemplar Theory", "Conceptual Spaces", "Theory-Theory", "Embodied Cognition").
theory_type str High‑level category of the representational format (e.g., "symbolic", "feature‑based vector", "probabilistic distribution", "high‑dimensional space", "relational network", "embodied simulation").
theory_description str A concise description of how the theory claims conceptual knowledge is functionally represented.
functional_claims str Key functional claims made by the theory (e.g., "accounts for typicality gradients", "explains category learning speed", "captures similarity judgments").
evidence_source str Type of empirical evidence cited (e.g., "behavioral experiment", "fMRI study", "EEG/MEG", "computational simulation", "lesion study").
experimental_paradigm str The experimental paradigm or task used to gather evidence (e.g., "typicality rating", "category verification", "similarity judgment", "semantic feature verification", "semantic priming", "neuroimaging semantic task").
key_result str A brief summary of the main finding relevant to the theory (e.g., "prototype effects observed", "exemplar similarity predicts reaction times", "distributed activation patterns correlate with feature vectors").
supports_theory bool Does the reported evidence support the theory? (true, false, or null if not explicitly stated).
counter_evidence str Any reported findings that challenge or contradict the theory's claims.
citation str Reference for the paper (authors and year) where the information was extracted.
