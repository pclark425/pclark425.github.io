Extraction Schema Details for extraction-schema-151

Extraction Schema (General Information)

Schema ID: extraction-schema-151

Extraction Query: Extract any mentions of LLM agents playing text games that use memory, including details of the memory mechanism, how it is integrated, performance results with and without memory, comparisons between memory strategies, and any recommendations or conclusions about the best use of memory.

Extraction Schema (Details)

Field Name Type Description
agent_name str The name of the LLM-based agent that plays the text game (e.g. 'MemAgent', 'ReAct', etc.).
agent_description str A brief description of the agent, including its main features and purpose.
llm_model_name str The name of the underlying LLM model used by the agent (e.g. 'GPT-3', 'GPT-4', 'PaLM', etc.).
llm_model_description str A brief description of the LLM model (e.g. size, architecture, notable features).
benchmark_name str The name of the text game or benchmark used for evaluation (e.g. 'TextWorld', 'Jericho', 'Zork', etc.).
benchmark_description str A brief description of the text game or benchmark, including task type and challenges.
memory_used bool Does the agent use a memory mechanism to solve the text game task? (true, false, or null if not specified)
memory_type str The type of memory used (e.g. episodic, working, external, retrieval-augmented, scratchpad, etc.).
memory_architecture str A description of the memory architecture (e.g. how memory is stored, updated, and accessed; e.g. key-value store, attention over history, etc.).
memory_integration_strategy str How the memory is integrated into the agent's decision-making (e.g. concatenated to input, retrieved via attention, used for planning, etc.).
performance_with_memory str The performance of the agent on the benchmark when using memory (numerical, include units, or null if not reported).
performance_without_memory str The performance of the agent on the benchmark without using memory (numerical, include units, or null if not reported).
has_performance_with_without_memory bool Does the paper report performance both with and without memory for this agent? (true, false, or null if not specified)
ablation_or_comparison str Any ablation studies or direct comparisons between different memory strategies, or between memory and no-memory baselines.
best_memory_strategy str If the paper identifies a memory strategy that works best, describe it briefly.
limitations_or_failure_cases str Any reported limitations, challenges, or failure cases related to memory usage in LLM agents for text games.
recommendations_or_conclusions str Any recommendations or conclusions from the paper about how LLM agents can best use memory to solve text game tasks.
