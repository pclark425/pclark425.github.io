Extraction Schema Details for extraction-schema-21

Extraction Schema (General Information)

Schema ID: extraction-schema-21

Extraction Query: Extract any mentions of agents that use external tools for planning in partially observable text environments, including how they maintain belief states, incorporate tool outputs, and perform navigation or path-finding tasks.

Extraction Schema (Details)

Field Name Type Description
agent_name str The name of the agent or model that performs planning with tools.
agent_description str A detailed description of the agent, including its architecture and key components.
environment_name str The name of the text environment or benchmark where the agent operates (e.g., TextWorld, ALFWorld, Jericho, etc.).
environment_description str A description of the environment, including whether it is partially observable and what makes it challenging.
is_partially_observable bool Is the environment partially observable (i.e., the agent cannot see the full state at once)? (true, false, or null for no information)
external_tools_used str What external tools does the agent use? List all tools mentioned (e.g., maps, calculators, search engines, knowledge bases, etc.). Be specific about each tool.
tool_output_types str What types of outputs do the tools provide? (e.g., spatial maps, numerical results, text snippets, structured data, etc.)
belief_state_mechanism str How does the agent maintain and update its belief state? Describe the mechanism in detail (e.g., memory networks, graph representations, textual summaries, etc.). null if no information.
incorporates_tool_outputs_in_belief bool Does the agent explicitly incorporate tool outputs into its belief state representation? (true, false, or null for no information)
belief_update_description str How are belief states updated when new information (including tool outputs) is received? Describe the update mechanism. null if no information.
planning_approach str What planning approach does the agent use? (e.g., model-based planning, search-based planning, learned policy, chain-of-thought reasoning, etc.)
uses_shortest_path_planning bool Does the agent perform shortest-path or navigation planning tasks? (true, false, or null for no information)
navigation_method str If the agent performs navigation, how does it compute or select paths? (e.g., A* search, learned navigation policy, graph-based planning, etc.). null if no information.
performance_with_tools str What is the performance of the agent when using external tools? (numerical, include units and task details. null if no information)
performance_without_tools str What is the performance of the agent without external tools (baseline)? (numerical, include units and task details. null if no information)
has_tool_ablation bool Does the paper report performance both with and without external tools? (true, false, or null for no information)
key_findings str What are the key findings about how tools, belief states, and planning interact? Summarize the main insights in a concise, information-dense way.
